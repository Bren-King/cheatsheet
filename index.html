Common CSS Text Properties

Selector Description Values
color Sets the text color Name, RGB, hex
line-height Sets the distance between text lines normal, number, percentage, length
letter-spacing Increase or Decrease space between letters normal, length, number, precentage
text-align Aligns text in an element left, right, center, justify
text-decoration Adds decoration to text none, underline, overline, line-through
text-transform Controls the letters in an element uppercase, lowercase, capitalize, none
text-indent Indents the first line for text lenght, percentage
